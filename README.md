# 运行与调试

```
cd bochs
./run.sh
```

此时会进入bochs虚拟机的BIOS代码，而我们的代码是从0x7C00位置开始的，所以先要在那里设一个断点，然后继续运行到断点：

```
<bochs:1> b 0x7c00
<bochs:2> c
<bochs:3> u /10  # 反汇编接下来10条指令，下面的指令与我们的asm代码一致
0000000000007c00: (                    ): jmpf 0x07c0:0005          ; ea0500c007
0000000000007c05: (                    ): mov ax, cs                ; 8cc8
0000000000007c07: (                    ): mov ds, ax                ; 8ed8
0000000000007c09: (                    ): mov es, ax                ; 8ec0
0000000000007c0b: (                    ): mov ss, ax                ; 8ed0
0000000000007c0d: (                    ): mov sp, 0x0400            ; bc0004
0000000000007c10: (                    ): mov dx, 0x0000            ; ba0000
0000000000007c13: (                    ): mov cx, 0x0002            ; b90200
0000000000007c16: (                    ): mov ax, 0x1000            ; b80010
0000000000007c19: (                    ): mov es, ax                ; 8ec0
```
